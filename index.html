Sapphire
Recursos Sintaxe Licença GitHub

Uma linguagem de programação com foco em performance e simplicidade.

Sapphire está em seu processo inicial de criação, com seu núcleo sendo desenvolvido em C++ para máxima velocidade e ferramentas em C# para uma experiência moderna.

Veja a Sintaxe

Uma Fundação Promissora

Combinando o poder de linguagens de baixo nível com a facilidade de scripts de alto nível.

Núcleo de Alta Performance

O interpretador principal de Sapphire é construído em C++, garantindo que seu código seja executado com eficiência e velocidade.

Arquitetura Híbrida

Sapphire é projetada com uma base dupla: o núcleo em C++ busca a performance, enquanto a interface e futuras ferramentas se beneficiarão da versatilidade do C#.

Sintaxe Clara e Expressiva

Projetada para ser fácil de ler e escrever. Nosso objetivo é ter menos código boilerplate e mais foco na sua lógica.

Conheça a Sintaxe

Veja um exemplo de como Sapphire lida com variáveis e operações matemáticas.

Sapphire Script 
// Declaração de variáveis
var a = 20
var b = 10
var c = 25

// Imprime o resultado da operação. O resultado esperado é -230 (20 - (10 * 25))
print a - b * c
